# Support

Thanks for using Re-Phraser. The GitHub repository is the home and support channel for this project.

## How to get help

Please open a ticket on **[GitHub Issues](https://github.com/atj393/local-rephraser/issues)**. Before opening a new issue, it helps to search existing issues in case your question has already been answered.

There is **no guaranteed response time**. This is a personal productivity project maintained on a best-effort basis.

## What to include in a bug report

A good report helps a lot. Please include:

- **Browser and version** (for example, Chrome 126 or Microsoft Edge 126)
- **Re-Phraser version** (see [CHANGELOG.md](CHANGELOG.md) or the extensions page)
- **The website** where the problem happened
- **What you expected** to happen
- **What actually happened**
- **Screenshots**, if helpful - with any sensitive content removed first

## Please do not share sensitive information

Re-Phraser works with your own text and your own AI chat session. When you write a report, **do not post**:

- Passwords or login details
- Private or confidential message content
- Your AI conversation URLs
- API tokens or secrets
- Anything you would not want to be public

Issues on GitHub are public. Remove or blur sensitive text in screenshots before uploading.

## Feature ideas

Feature requests and ideas are welcome through GitHub Issues as well. Please describe the problem you are trying to solve, not only the solution you have in mind.

## Security reports

Please do not report security vulnerabilities in public issues. See [SECURITY.md](SECURITY.md) for how to report them.
